To run the files, ensure that you are in the correct directory in your terminal:
cd <project_folder_path>

Input parameters n, numOps, rndLt and lambda are present in inp-params.txt.

To run CLQ1, execute:
g++ CLQ1-ES20BTECH11029.cpp -o CLQ1
./CLQ1

The output will be written in CLQ1-out.txt.

To run NLQ1, execute:
g++ NLQ1-ES20BTECH11029.cpp -o NLQ1
./NLQ1
The output will be written in NLQ1-out.txt.
