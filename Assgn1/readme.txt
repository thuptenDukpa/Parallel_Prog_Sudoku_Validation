To run the files, ensure that you are in the correct directory in your terminal:
cd <project_folder_path>

Input parameters n and m are present in inp-params.txt.

To run DAM1, execute:
g++ DAM1-ES20BTECH11029.cpp -o DAM1
./DAM1

The output will be written in Primes-DAM1.txt.

To run SAM1, execute:
g++ SAM1-ES20BTECH11029.cpp -o SAM1
./SAM1
The output will be written in Primes-SAM1.txt.
